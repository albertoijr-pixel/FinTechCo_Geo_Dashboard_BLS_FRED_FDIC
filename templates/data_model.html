Layer 1 — Raw Data Sources
BLS
Bureau of Labor Statistics
api.bls.gov/publicAPI/v2/timeseries/data/
🔑 API key required  ·  OEWS state-level wage series
Raw fields returned
state_fips state_name naics_code sector_name mean_annual_wage
Series pulled (SOC → NAICS proxy)
SOC 15-0000 → Tech (5415) SOC 29-0000 → Healthcare (6211) SOC 23-0000 → Legal (5411)

FDIC
Federal Deposit Insurance Corp.
banks.data.fdic.gov/api/financials
🔓 No auth required  ·  Annual call report data
Raw fields returned
STNAME → state DEP → total_deposits ($K→$M) DEPNI → nonint_deposits ($K→$M) CERT → institution_count
Query parameters
REPDTE: 2023-12-31 agg_by: STNAME agg_sum: DEP, DEPNI agg_count: CERT
Coverage
51 states + DC Year-end 2023 Base table for join
FRED
St. Louis Federal Reserve
api.stlouisfed.org/fred/series/observations
🔑 API key required  ·  Macro time-series
Series pulled
DPSACBW027SBOG date deposits_billions
Nature of data
National aggregate only Weekly, seasonally adj. No state breakdown
⚠ Defined but not active in scorer. fetch_fred_deposits() is implemented in data_fetcher.py but compute_scores() does not call it — FRED is not currently joined into the per-state model. Candidate for a future national normalisation pass.
